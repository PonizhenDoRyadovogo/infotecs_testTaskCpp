# LRU Cache Project

Реализация многопоточного кэша на основе алгоритма **LRU (Least Recently Used)**. Проект выполнен в рамках тестового задания.

## Особенности реализации
- **Производительность:** Операции `put` и `get` выполняются за константное время $O(1)$.
- **Потокобезопасность:** Доступ к данным синхронизирован с помощью `std::mutex`.
- **Гибкость:** Использование шаблонов (templates) позволяет хранить любые типы данных.
- **Тестирование:** Включены автоматизированные тесты (unit-tests) и проверка работы в многопоточной среде.

## Системные требования
* **C++ стандарт:** C++17 или выше.
* **Система сборки:** CMake 3.10+.
* **Компилятор:** GCC, Clang или MSVC.
